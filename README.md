# WeChat Publisher

> 在 Obsidian 里写作，一键发布到微信公众号。

WeChat Publisher 是一个 Obsidian 插件，让你无需离开 Obsidian 就能完成公众号排版和发布。

---

## 功能

### 免费功能

- **格式预览**：Markdown 实时渲染为微信样式，所见即所发
- **复制 HTML**：一键复制渲染结果，粘贴到公众号编辑器
- **跳转公众号平台**：快速打开微信公众号后台

### 付费功能

- **发布到草稿箱**：直接推送到公众号，无需打开浏览器
- **账号配置**：直连公众号 API，支持多账号管理
- **封面系统**：自动设置封面，支持默认封面

### 支持的格式

| 格式 | 说明 |
|------|------|
| 代码块 | 语法高亮，保留缩进与空格 |
| 数学公式 | `$...$` 行内 / `$$...$$` 块级，渲染为图片 |
| Mermaid 图表 | 流程图、时序图等，渲染为图片 |
| 表格 | 完整样式还原 |
| 图片 | 本地 + 远程图片，自动上传微信 CDN |
| Ruby 注音 | `{文字\|注音}` 语法 |
| 警示块 | `> [!NOTE]` / `[!WARNING]` 等 |
| 横向滑图 | 多图并排轮播 |

## 用户指南

WeChat Publisher 目前尚未上架 Obsidian 官方插件市场，提供两种安装方式：

| 方式 | 适合人群 | 特点 |
|------|----------|------|
| **方式一：BRAT 安装（推荐）** | 所有用户 | 自动检测更新，一键升级 |
| **方式二：手动安装** | 熟悉文件操作的用户 | 无需额外插件 |

---

### 方式一：通过 BRAT 安装（推荐）

BRAT 是 Obsidian 官方市场中的一个插件管理工具，可以直接从 GitHub 安装未上架的插件，并自动管理更新。

**第一步：安装 BRAT**

在 Obsidian「设置」→「第三方插件」→「浏览」中搜索 **BRAT**，安装并启用。
![PixPin_2026-03-27_11-13-12](https://cf-img.discoverlabs.ac.cn/202603272fe616a63272ecc499e14c731934ac9f.webp)

**第二步：添加 WeChat Publisher**

打开「设置」→「BRAT」，点击 **Add Beta plugin**，输入仓库地址：

```
RanceLee233/wechat-publisher
```

点击 **Add Plugin**，选择 Latest version，BRAT 会自动下载安装最新版本。
![PixPin_2026-03-27_11-13-56](https://cf-img.discoverlabs.ac.cn/2026032713d6d729fcf1e57760c4cbf371763d3c.webp)
![PixPin_2026-03-27_11-19-04](https://cf-img.discoverlabs.ac.cn/20260327470a731d145a98e625b8ed39b6ecb971.webp)

**第三步：启用插件**

回到「设置」→「第三方插件」，找到 **WeChat Publisher**，开启开关启用。

> **后续更新**：BRAT 会自动检测新版本。也可以在「设置」→「BRAT」中手动点击 **Check for updates** 立即检查。

---

### 方式二：手动安装

**第一步：下载插件**

前往 [博客下载页](https://blog.discoverlabs.ac.cn/downloads/wechat-publisher/) 下载最新版本 zip（国内直连），或前往 [GitHub Releases](https://github.com/RanceLee233/wechat-publisher/releases) 下载历史版本。

**第二步：解压到 Obsidian 插件目录**

将 zip 解压后，把文件夹复制到 Obsidian 插件目录：

```
你的仓库路径/.obsidian/plugins/wechat-publisher/
```

> **如何找到插件目录？** 在 Obsidian「设置」→「第三方插件」中点击「打开插件文件夹」即可定位。

![PixPin_2026-03-27_10-52-37](https://cf-img.discoverlabs.ac.cn/20260327e50f0cd87e0dff2b55587f5e26f287f4.webp)

完成后目录结构应为：

```
.obsidian/
  plugins/
    wechat-publisher/
      main.js
      manifest.json
      styles.css
```

**第三步：启用插件**

打开「设置」→「第三方插件」，找到 **WeChat Publisher**，点击右侧开关启用。

![PixPin_2026-03-27_10-53-46](https://cf-img.discoverlabs.ac.cn/20260327e97ca2137887b2088bd22fd595206657.webp)

### 免费功能

#### 格式

点击顶部的**格式**按钮，可以展开样式控制栏，对排版进行调整。

样式控制栏从左到右包含：

| 按钮 | 说明 |
|------|------|
| **主题** | 切换整体主题风格 |
| **经典蓝** / **专栏版** 等 | 内置预设主题，点击直接应用 |
| **样式** | 展开细节样式调节选项 |
| **已存样式** | 查看你保存过的自定义样式 |
| **套用已存样式** | 将已保存的样式应用到当前文章 |
| **样式配置** | 进入详细样式参数调节面板 |

**建议：** 首次使用时先在「样式配置」里调好你偏好的排版参数，然后用「已存样式」保存，之后写每篇文章只需点「套用已存样式」即可。

![PixPin_2026-03-26_19-27-26](https://cf-img.discoverlabs.ac.cn/20260327cc923f87a73d4ce0f348f02a55134e60.webp)

#### 复制 HTML

点击顶部的**复制 HTML** 按钮，插件会将当前预览区的渲染结果复制为微信兼容的 HTML。

复制完成后，打开微信公众号后台编辑器，在正文区域直接粘贴即可。适合不需要直连 API、手动上传文章的场景。

---

#### 跳转公众号平台

点击**跳转公众号平台**，浏览器会自动打开微信公众号后台 `mp.weixin.qq.com`，方便你切换到后台进行后续操作。

### 付费功能

付费功能需要先完成**账号配置**和**激活码绑定**才能使用。

---

#### 账号配置

点击顶部操作栏中的**账号配置**按钮，打开配置弹窗。

![PixPin_2026-03-24_23-11-55](https://cf-img.discoverlabs.ac.cn/2026032724e0825e42bccb6bc54e4d7a4ab2857b.webp)

弹窗中各字段说明如下：

![PixPin_2026-03-24_23-12-26](https://cf-img.discoverlabs.ac.cn/20260327832b93b88ca69d7f9695d7fe56a402c0.webp)

| 字段 | 说明 |
|------|------|
| **账号名称** | 自定义名称，方便区分多个公众号，例如：主号 / 备用号 |
| **AppID** | 微信公众号的 AppID，在微信公众平台「基础信息」页获取 |
| **AppSecret** | 微信公众号的开发密钥，需在微信公众平台手动启用 |
| **IP 白名单辅助** | 插件自动检测你当前的公网 IP，点击「复制」后去微信平台填入白名单 |
| **默认作者** | 可选。发布资料区没有填写作者时，自动使用此处的名字 |
| **默认封面** | 可选。为该账号设置一张固定封面，每次发文章时自动使用 |
| **设为默认账号** | 开启后，打开插件时自动选中该账号 |

底部按钮：
- **新增账号**：添加另一个公众号账号
- **保存**：保存当前账号的配置

> AppSecret 仅保存在本地 Obsidian 配置文件中，不会上传到任何服务器。

**第一步：** 打开微信开发者平台 `developers.weixin.qq.com`，点击**前往控制台**。

![PixPin_2026-03-24_23-03-40](https://cf-img.discoverlabs.ac.cn/20260327338a1f10db1b76c8dfb8661961a2b5ac.webp)

**第二步：** 在「我的业务」页面找到你的**公众号**并点击进入。

![PixPin_2026-03-24_23-04-16](https://cf-img.discoverlabs.ac.cn/202603279344ceddd894874ea59b66cf38eca2d4.webp)

**第三步：** 进入公众号管理后台，在**基础信息**页面找到 **AppID**（直接复制）以及**开发密钥**区域。点击 AppSecret 旁的**启用**，扫码授权后即可获取 AppSecret。

![PixPin_2026-03-24_23-08-15-1](https://cf-img.discoverlabs.ac.cn/20260327b8996df79a73ea68add38c2e8b883605.webp)

> AppSecret 只会在启用时完整显示一次，请立即复制保存。

##### 如何配置 IP 白名单

微信公众号 API 要求调用接口的服务器 IP 在白名单内。由于 WeChat Publisher 从你的电脑直接调用 API，需要将你的电脑公网 IP 加入白名单。

**操作步骤：**

1. 在插件的账号配置弹窗中，点击 **检测本机 IP**，插件自动获取你的当前公网 IP
2. 点击**复制**，复制该 IP 地址
3. 点击 **IP 设置**，或回到微信公众平台的「基础信息」页，点击「API IP 白名单」旁的**设置名单**
4. 将复制的 IP 粘贴进去，多个 IP 用回车隔开，点击**确定**

![PixPin_2026-03-24_23-11-04](https://cf-img.discoverlabs.ac.cn/202603276a9d7ae46c43ba51beae9c7f90a1754c.webp)

> **注意：** 如果你在不同网络环境下使用（如家里和公司），每个网络的公网 IP 不同，需要分别添加。

---

#### 激活码绑定

使用付费功能前，需要将激活码绑定到你的公众号 AppID。

**绑定步骤：**

1. 打开**账号配置**弹窗，在下方「账号授权」区域，将激活码填入输入框
2. 确认当前账号的 AppID 已正确填写
3. 点击**绑定当前公众号**

![PixPin_2026-03-26_11-42-25](https://cf-img.discoverlabs.ac.cn/20260327835f4c00ba78a5dfc50d3f1d048bef8d.webp)

绑定成功后，界面会弹出提示「已绑定到当前公众号」，授权状态区域同步显示绑定时间和最近校验时间。

![PixPin_2026-03-26_11-51-40](https://cf-img.discoverlabs.ac.cn/2026032762404f3e91d91b254ab9489a0af2f758.webp)

**重要说明：**
- 1 个激活码只能绑定 1 个公众号 AppID，绑定后不支持改绑
- 如果你换了新电脑，不需要重新输入激活码，点击**刷新授权状态**，插件会自动识别已绑定的 AppID 并恢复授权
- 免费版状态下，已配置的账号信息会保留，但发布等付费功能不可用

> **数据说明：** 点击绑定时，插件会将**激活码 + AppID** 发送到授权服务器进行绑定记录。AppSecret 始终只保存在本地，不会发送到任何服务器。

---

#### 发布草稿

完成账号配置和激活码绑定后，点击顶部的**发布草稿**按钮。

插件会自动：
1. 将文章中的本地图片上传到微信 CDN
2. 处理 Mermaid 图表、数学公式等为图片
3. 将渲染后的 HTML 连同标题、作者、封面一起提交到公众号草稿箱

发布成功后，登录微信公众号后台，在**草稿箱**中即可找到该文章，审阅无误后再群发。

> 草稿发布后不会直接群发，需要在公众号后台手动操作群发。

---

#### 封面系统

封面设置位于发布资料区，包含三个选项：

![PixPin_2026-03-26_19-28-03](https://cf-img.discoverlabs.ac.cn/20260327c9d6037db80885c6f2fe738fc425bb77.webp)

| 按钮 | 说明 |
|------|------|
| **从电脑选择封面** | 打开文件选择器，从本地选一张图片作为本次文章的封面 |
| **清空封面** | 清除当前设置的封面，发布时使用占位空白封面 |
| **默认封面** | 使用账号配置中设定的默认封面（需在账号配置中预先设置） |

**封面优先级：** 手动选择的封面 > 默认封面 > 占位封面（空白）

在**账号配置**弹窗中也可以管理默认封面：
- **重新选择默认封面**：更换该账号的默认封面图片
- **清空默认封面**：删除已设置的默认封面

> 封面图片建议尺寸为 900×383 像素（微信公众号推荐比例），格式支持 JPG、PNG。

- 激活码绑定时传输：**激活码 + AppID**（用于绑定公众号），不含个人信息；AppSecret 始终不离开本地
- 图片直接上传至微信官方 CDN，不经过第三方服务器
